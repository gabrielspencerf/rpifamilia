# rpifamilia

Landing page **Roland, Pessalli & Isidoro** — **Direito de Família** (Porto Alegre e Região Metropolitana). React, Vite, TypeScript e Tailwind. Chat via Typebot (`rpifamilia`).

Repositório: [github.com/gabrielspencerf/rpifamilia](https://github.com/gabrielspencerf/rpifamilia)

## Rodar localmente

**Requisito:** Node.js

```bash
npm install
npm run dev
```

Abre em [http://localhost:3000](http://localhost:3000) (porta definida no Vite).

## Build

```bash
npm run build
npm run preview
```
